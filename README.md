# Subconscious Study

**EN**





**Learning on autopilot**



Subconscious Study loads your notes, study materials or any text file and quietly surfaces small chunks of it throughout your day — no active studying required.



##### How it works

1. Load your .txt, .md, .pdf or .docx files
2. Set how often you want random popups (I suggest high intervals to avoid overload)
3. Click start and go do something you like. Chunks of infos will appear and you can give them a fast read.
4. Move and resize popups to fit your workspace better. The next popups will have the same size and position you set.



##### Download

[**→ Download the latest version**](https://github.com/Rcipolletta/SubconsciousStudy/releases/latest)

Windows only (for now, unless asked for other OSs).



##### Features

* Supports .txt, .md, .pdf or .docx files
* Popups can be set to be always on top or not
* The app can be set to start at Windows startup



##### Requirements

Windows 10 or later.

**IT**





**Tu fai altro, il tuo cervello memorizza**



Subconscious Study ti permette di caricare appunti, dispense e file di testo per mostrarti piccoli popup durante la giornate. Questo stile popup pubblicitario ti aiuta a memorizzare o ricordare informazioni **senza che tu debba studiare attivamente.**



##### Download

[**→ Scarica qui l'ultima versione**](https://github.com/Rcipolletta/SubconsciousStudy/releases/latest)

Solo Windows (Per ora, a meno che non ci sia richiesta per altri OS).





##### Come funziona

1. Carica i tuoi file in formato .txt, .md, .pdf or .docx
2. Scegli l'intervallo di tempo tra un popup e un altro (raccomando intervalli maggiori di 5 minuti per evitare il sovraccarico mentale)
3. Clicca su "Avvia sessione" e vai a fare qualcos'altro
4. Sposta e ridimensiona i popup così che non siano davanti alle tue attività: l'app memorizza queste informazioni e il prossimo popup sarà nella posizione e dimensione che hai scelto



##### Caratteristiche

* Supporta file in formato .txt, .md, .pdf or .docx
* Puoi scegliere se i popup restano sempre davanti alle altre finestre e applicazioni
* Puoi impostare l'app in modo che si avvii all'avvio di Windows



##### Requisiti

Windows 10 o successivi.
